# SHA-256 Hash Generator

**Курсовая работа • Реализация алгоритма SHA-256 с нуля на C++ + GUI на Qt**

## О проекте

Полноценная реализация криптографического алгоритма **SHA-256** «с нуля» без использования сторонних библиотек.  
Проект включает:

- Реализацию алгоритма **MD5** (для понимания принципов хеширования)
- Полную реализацию **SHA-256** согласно спецификации NIST FIPS 180-4
- Графический интерфейс на **Qt** для удобного ввода текста и получения хеша
- Тестирование и сравнение результатов с эталонными значениями (Python `hashlib` + онлайн-сервисы)

Работа выполнена в рамках курсовой по дисциплине «Структуры данных и алгоритмы» в НИУ ВШЭ (МИЭМ).

## Основные возможности

- Хеширование произвольного текста (включая кириллицу)
- Чистая реализация SHA-256 (без `OpenSSL`, `Crypto++` и т.д.)
- Интуитивно понятный GUI на Qt
- Побитовые операции, дополнение сообщения, 64 раунда сжатия
- Сравнение результатов с эталонными значениями

## Технологии

- **Язык:** C++17
- **GUI:** Qt 6
- **Сборка:** CMake / Qt Creator
- **Платформы:** Windows / Linux / macOS
